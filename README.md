# Tool-for-extracting-highlighted-vocabulary-from-text
📚✒️This tool allows you to extract vocabulary highlighted in blue from text, suitable for language learners who need to extract vocabulary from images and text.
Instructions for use:
1. Send the image file containing the text and the highlighted vocabulary to the website.
2. Click the "Scan Vocabulary" button.
3. The website will return a list of vocabulary. If any words are missing or not highlighted, you can double-click on the word in the image, and the website will manually add it to the end of the list.
4. Click the "Copy" button to copy the vocabulary list.

📚✒️Công cụ này giúp cho phép bạn có thể lấy từ vựng được đánh dấu bôi xanh trên văn bản, phù hợp cho các bạn học ngoại ngữ cần trích xuất từ vựng trên hình ảnh văn bản của bạn.
Các bước sử dụng:
1. Gửi file hình ảnh chứa văn bản và các từ vựng đã được đánh dấu (highlight bằng bút dạ quang) vào web.
2. Nhấn nút quét từ vựng.
3. Web trả về danh sách từ vựng, trong trường hợp bị quét thiếu, hoặc có các từ chưa được highlight, bạn hoàn toàn có thể click đúp vào từ vựng trên ảnh đó, web sẽ tự thêm thủ công vào cuối danh sách cho bạn.
4. Nhấn nút copy để copy danh sách từ vựng.
